Important FAQ

What is Document Object Model ( DOM ) ?

The Document Object Model (DOM) is a programming interface representing an HTML/XML document as a tree structure, enabling dynamic interaction via JavaScript.

How do you select on element form DOM ?

To select an element from the DOM, use methods like document.getElementById("id"), document.querySelector("selector"), or document.getElementsByClassName("class").

What is event delegation in the contex of the DOM , and why is it useful ?

Event delegation is a technique where a single event listener is added to a parent element to handle events on its child elements. This works because events bubble up from child to parent in the DOM. It improves performance by reducing the number of event listeners, especially for dynamically added elements.

How do you manipulate an element's attributes and styles using the DOM ?

To manipulate an element's attributes and styles using the DOM, you can use methods like setAttribute and getAttribute to modify attributes such as id, src, or title. For styling, you can access the style property and change CSS styles directly. For example, element.style.color = "red"; sets the text color to red, and element.setAttribute("title", "New Title"); updates the element's title attribute.
